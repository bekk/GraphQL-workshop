# GraphQL-workshop

# Installere på forhånd
* node / yarn + npm

## Backend:

`yarny`

`node GraphqlServer.js `

Vipps, du har en server kjørende på [localhost:4000](http://localhost:4000/)

## Frontend:
                                   
`cd client`

`yarn && yarn start`

Dobbelvipps, du har en klient kjørende på [localhost:3000](http://localhost:3000)
